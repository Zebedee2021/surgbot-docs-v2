# 项目简介

## 背景

本项目由**雄安宣武医院神经外科**（首都医科大学宣武医院雄安院区）与**北京理工大学具身空间智能实验室**（Embodied Space & AI Lab）合作开展。

手术室中，器械护士（又称刷手护士）承担着手术器械管理与传递的关键职责，工作强度高、对精准度要求严格。本项目旨在研发一款**手术器械护士机器助手**，在神经外科手术室场景中辅助器械护士完成高频、标准化的器械操作任务。

## 项目目标

1. **器械识别**：精准识别手术器械的种类、位置和朝向
2. **智能抓取**：机械臂自动完成器械的稳定抓取
3. **规范递交**：按照手术规范将器械递交给医生（把手朝向医生）
4. **语音交互**：通过语音指令触发器械递交流程
5. **安全可靠**：完善的安全策略，支持急停和异常接管

## 核心定位

> **人机协同，而非替代。**

机器助手定位为器械护士的**辅助**，处理约 70–80% 的高频标准器械递交任务。器械护士保留决策权和异常处理能力，机器助手通过**触发 → 确认 → 交接**的工作流与护士形成配合。
